# Smart-Slip-Analyzer
This project create for study experience to making full-stack development

## Website

Smart Slip Analyzer is a static AI personal finance web app prototype. It includes:

- Slip upload and demo AI scan interaction
- Extracted merchant, amount, category, and date fields
- Income, expense, savings, and balance dashboard metrics
- Spending category analytics
- Detected transaction list
- AI financial recommendations

Open `index.html` in a browser to run the website.
